Adaugarea folderului RSS/ATOM din linia de comanda

Created At：2023-10-25 10:33 Updated At：2023-12-12 13:40
Keywords: Admin Zimbra

Adaugarea din CLI

zmmailbox -z -m user@tld.md cf -u 'https://blog.zimbra.com/feed/' '/Zimbra Blog'
zmmailbox -z -m user@tld.md cf -u 'https://wiki.zimbra.com/security-advisory-feed.php' '/Zimbra Security'

In Zimbra 10 Modern UI este Zimmlet


wget https://github.com/Zimbra/zimbra-zimlet-rssfeed/releases/download/0.0.1/zimbra-zimlet-rssfeed.zip -O /tmp/zimbra-zimlet-rssfeed.zip

sudo su - zimbra
zmzimletctl deploy /tmp/zimbra-zimlet-rssfeed.zip
zmmailboxdctl restart

Sursa: https://blog.zimbra.com/2023/10/re-introducing-rss-feeds-in-zimbra/
