Sincronizarea contactelor globale din Zimbra cu OUTLOOK

1. Descărcăm pluginul Outlook CalDav Synchronizer de pe siteul oficial - caldavsynchronizer.org

Versiunea actuala este 4.0.0

2. Se rulează programul de instalare din arhiva descărcată

3. Instalarea Outlook CalDav Synchronizer

4. Repornim clientul MS Microsoft Outlook

5. Următorul pas este configurarea profilelor pentru contacte

6. Se adaugă un profil “Generic ValDAV/CardDAV”

7.  Adăugați profilul cu umatoarele date:

  • DAV URL: https://mail-domeniu.gov.md/dav/gal@domeniu.gov.md/_gal

  • Username: gal@domeniu.gov.md

  • Password: XXXXXXXX

  • Email: gal@domeniu.gov.md

  • Alegeți parametrul de sincronizare Outlook ß Server

Scrieti denumirea profilului si alegeți folderul unde doriti sa fie stocate contactele (Implicit este folderul Contacts)

8. După aceste setări apăsați butonul “Test or discover settings” si alegeți “_gal”

9. In caz ca apare eroarea de mai jos, acceptați, si continuați cu “OK”

10. In scurt timp dupa sincronizare, in lista de contacte vor apărea toate adresele e-mail si listele de distribuție

 

NOTE:

1. In unele cazuri in Outlook, addon-ul  CallSav Sincronizer este dezactivat după instalare, in acest caz este necesar de al active in setările Outlook (File/Options/Add-INS)  , in compartimentul Disables Application Add-Ins acesta poate fi activat

2.  Aplicatia Outlook CalDav Synchronizer   poate fi folosita pentru sincronizarea calendarului si altor entitati oferite de Sistemul de posta electronica (Briefcase, Tasks)
