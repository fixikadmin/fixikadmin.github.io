Cum se creează un token de acces personal, jetoane API personale în Github

Când am introdus codul pe GitHub, arată că nu aveți Token de acces personal. Când încerc să împing cod pe github, arată metoda cURL de autentificare (crearea unui token de acces personal). Am încercat asta, dar încă nu pot apăsa pe GitHub.

Deci haideți să rezolvăm aceste probleme, sper să vă fie de mare ajutor.

Primul pas Conectați-vă la Github și accesați secțiunea Setare.

2. În bara laterală din stânga, faceți clic pe Setări pentru dezvoltatori.

Al treilea pas În bara laterală din stânga , faceți clic pe Jetoane de acces personale.

Pasul 4 Oferiți control complet asupra depozitelor private.

Selectați domeniile sau permisiunile pe care doriți să le acordați acest simbol. Pentru a utiliza simbolul pentru a accesa depozitele din linia de comandă, selectați repo.

Pasul 5 Faceți clic pe Generare token nou

Acum, jetonul dvs. personal de acces a fost generat cu succes.

Pentru a seta configurația globală a numelui de utilizator/e-mail:

Deschideți linia de comandă.
Setați-vă numele de utilizator:
git config - global user.name "laravelamit"
Setați adresa de e-mail:
git config - global user.email "amitseo72@gmail.com"

Acum, toate configurațiile și simbolurile generate cu succes, acum puteți extrage și împinge cod pe GitHub.

Sper să vă fie de ajutor. 👍👍
