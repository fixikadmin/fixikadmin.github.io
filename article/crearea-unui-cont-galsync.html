Crearea unui cont GALSYNC

1. Se creaza contul e-mail

zmprov ca galsync@tld.md PaSSw0RD

2. Se adauga Data source

zmgsautil addDataSource -a galsync@tld.md -n InternalGAL --domain tld.md -t zimbra -f zimbra -p 60m

3. Se cauta DATA SOURCE ID
zmprov gds galsync@tld.md

4. Se modifica Domenul

zmprov modifyDomain tld.md fa0362b6-7e7a-4463-9214-9af047f03f86
